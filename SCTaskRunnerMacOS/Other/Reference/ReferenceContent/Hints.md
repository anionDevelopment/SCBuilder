# Hints

## Requirements

The following tools are required to build this code-unit:

- `git`
- `gitversion`
- `python`
- `scriptcollection`

To create a release the following tools are also required:

- `gh`
- `twine`

## IDE

The recommended IDE for this code-unit is [Visual Studio Code](https://code.visualstudio.com/).

It is recommended to enable word-wrap.
